# Stock Management Assistant

Manual Codex agent for checking the user's current portfolio and finding catalyst-backed stock ideas.

Run this agent manually whenever a stock check is needed. It returns the result in the current Codex chat.

## Files

- `AGENTS.md`: operating rules for Codex
- `prompt.md`: paste-ready manual prompt
- `config/portfolio.md`: current holdings and trading preferences
- `config/research-policy.md`: research and recommendation rules
- `memory.md`: local notes for future manual runs
